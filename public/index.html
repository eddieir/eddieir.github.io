Master thesis

I have done my master thesis at RWTH University of Aachen with the title of "Energy Efficient Data Center Operation: Measurements and Analysis". This master thesis was divided into two phases: prediction and optimization.
First phase: Prediction of consumed power inside the data center environment and this phase was implemented by using Python programming language and ARIMA time-series forecasting. The results of this phase were 99% accuracy.
Second phase: Optimization of the overall consumed power inside the data center environment, this phase also done by Python programming language with the help of Grubi solver and the result of this phase reducing 13KW of consumed power.
